# Classroom
Project cuối kỳ môn "Phát triển hệ thống thông tin doanh nghiệp"

Cách chạy project:
 + Cài đặt Django
 + Cài đặt mySQL (XAMPP)
 + Import database classroom.sql trong folder "database"
 + Mở CMD tại thư mục project
 + gõ: python manager.py runserver
